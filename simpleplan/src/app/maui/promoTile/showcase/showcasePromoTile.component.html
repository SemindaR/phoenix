The MAUI Promo Tile is used to present promotional offers

The tile has 4 sections, the header, body, footer(link) and the image.

The body section of the tile is passed and content between the tags (tranclusion) so that html formatting can be used.

Note: Image sizes are set to 130 x 130 px and 70 x 70 for mobile.

Attributes
header string Required
footerLinkText string Required
footerLink string Required
imagePath string Required
showTile boolean Required
